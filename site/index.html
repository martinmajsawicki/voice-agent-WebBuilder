About the Creator

Marcin Sawicki is a television journalist with interests in technology and applications of generative artificial intelligence. He created this tool as a prototype for voice-driven content generation.

Technical Architecture

The tool integrates multiple components to enable voice-controlled web page generation:

  • Eleven Labs voice agent for speech processing
  • Function Calling via Webhook for API communication
  • Python Flask microserver for backend processing
  • ngrok for secure tunneling
  • Anthropic API (Claude Sonnet) for content generation
  • GitHub repository for version control
  • Vercel for deployment
  • Mobile browser integration for photo uploads

Use Cases

This prototype addresses the needs of professionals who have information in their minds and photos on their phones, but lack time to work at a computer. Voice replaces the keyboard, and the AI agent replaces manual programming.

Real Estate Agent: Dictating property listings while on-site
Event Organizer: Creating live reports during events
Sales Representative: Updating content between meetings
