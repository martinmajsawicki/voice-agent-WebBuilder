Jak działa narzędzie

System pozwala na generowanie treści stron internetowych wraz ze zdjęciami poprzez rozmowę z agentem głosowym. Użytkownik opisuje zawartość strony mówiąc, a agent AI tworzy gotową stronę HTML.

Proces działania: użytkownik prowadzi rozmowę głosową z agentem AI, który analizuje przekazane informacje i automatycznie generuje strukturę oraz treść strony internetowej. Dodatkowo możliwe jest dodawanie zdjęć w czasie rzeczywistym przez interfejs mobilny.

Architektura techniczna

  • • Agent głosowy z function calling połączony przez webhook
  • • Flask Python Micro Server do obsługi żądań
  • • ngrok do tunelowania połączeń
  • • Cloud Sonnet łączący się przez Anthropic API
  • • Repozytorium GitHub do przechowywania kodu
  • • Vercel z automatycznym wdrażaniem
  • • Przeglądarka telefonu do uploadu zdjęć przez stronę mobilną
